Comment configurer les places?
  1. Selectionnez un voyage dans la liste ci-dessus
  2. Ajoutez les types de places disponibles (Premium, Standard, Economique)
  3. Definissez le nombre de places pour chaque type
  4. Definissez le prix unitaire pour chaque type
  5. Sauvegardez la configuration
Le total des places configurees ne doit pas depasser la capacite du vehicule.
